ztxt 3.3

Built by Zevan Rosser

ztxt is a browser based code editor. For use with HTML, CSS, JavaScript and PHP. ztxt works best with Safari or Google Chrome.

Watch the video tutorial

Credits

ztxt uses of a few excellent JavaScript libraries:

jQuery - for writing less and doing more
CodeMirror - for syntax highlighting and editing
jsBeautifier - to format .js files
Zen Coding - for high speed html/css coding
jQuery Color Picker - for choosing hex colors
jQuery File Tree - for file icons and css

ztxt was partially inspired by the excellent in-browser editor jsFiddle
